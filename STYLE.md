# Style guide


## XML

```xml
<!--
  Multi-line comment.
  Blah blah blah.
-->

<!-- One-liner -->

<FitsOnOneLine attribute="value">content</FitsOnOneLine>

<NeedsMultipleLines
  attribute1="value1"
  attribute2="value2"
  attribute3="value3"
>
  <SelfClosingElement
    attribute1="value1"
    attribute2="value2"
    attribute3="value3"
  />
  <AnotherSelfClosingElement
    attribute1="value1"
    attribute2="value2"
    attribute3="value3"
  />
</NeedsMultipleLines>
```
