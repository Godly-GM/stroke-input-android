# Style guide


## XML

```xml
<!--
  Multi-line comment.
  Blah blah blah.
-->

<!-- One-liner -->

<OneLiner attribute="value">content</OneLiner>
<SelfClosingOneLiner attribute="value" />

<MultiLiner
  attribute1="value1"
  attribute2="value2"
  attribute3="value3"
>
  <SelfClosing1
    attribute1="value1"
    attribute2="value2"
    attribute3="value3"
  />
  <SelfClosing2
    attribute1="value1"
    attribute2="value2"
    attribute3="value3"
  />
</MultiLiner>
```
